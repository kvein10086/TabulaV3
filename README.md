## :sparkles: 功能介绍 (Features)

### 基础功能

* 🃏 **卡片式整理**：Tinder 风格的卡片交互，左右滑动保留、上滑删除、下滑归类，单手即可轻松操作
* 📦 **批次管理**：每次推送 5/10/15/20/30 张照片，利用碎片时间轻松整理，拒绝焦虑
* 🗑️ **智能回收站**：删除照片先进入本地回收站，支持批量恢复，给你后悔的机会
* 🔍 **全屏查看器**：双指缩放（1-8x）、双击放大、HDR 对比、Live 照片播放

### 图集管理

* 📁 **自定义图集**：创建专属图集，自定义名称、颜色、图标，拖拽排序
* ⬇️ **下滑归类**：下滑卡片直接归入指定图集，搭配 Genie 收缩动画
* 🔄 **同步系统相册**：一键将图集同步到系统相册，支持复制/移动两种模式
* ↩️ **撤销操作**：支持撤销最近 10 次图集操作

### 智能推荐

* 🎲 **随机漫步**：真正随机抽取照片，带冷却期机制避免重复
* 🔗 **相似推荐**：基于时间、尺寸、大小等特征，优先推荐相似照片帮你清理连拍

### 个性化设置

* 🎨 **主题切换**：跟随系统 / 浅色 / 深色 三种模式
* 📊 **顶部显示**：可选索引模式（1/15）或日期模式（Jan 2026）
* 🖼️ **卡片样式**：固定比例（3:4）或自适应原图比例
* 📳 **振动反馈**：全局振动开关、强度调节（0-100）、滑动振动
* 🔊 **声音控制**：Live 照片声音开关及音量调节

### 特色体验

* 🌪️ **预测性返回**：完美适配 Android 15，对标 ColorOS 16 的丝滑过渡动画
* 🐈‍⬛ **小猫咪彩蛋**：可爱的 Tabula 小黑猫陪伴你整理照片，完成任务更有庆祝动画
* 📊 **整理统计**：累计整理数、删除数、剩余数量，进度一目了然
* 🔔 **桌面小组件**：显示待整理数量，随机激励语，点击直达
* ☁️ **模拟流体云**：退出应用时状态栏显示剩余照片数量（实验功能）

### 隐私保护

* 🔒 **100% 本地处理**：所有数据存储在本地，绝不上传任何照片
* 🙈 **无网络权限**：应用不申请联网权限，隐私彻底保障
* 🛡️ **安全删除**：使用系统 API 安全删除，可靠有保障
